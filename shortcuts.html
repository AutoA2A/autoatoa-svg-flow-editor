快捷键指南

以下列出交互式流程图编辑器中常用的键盘快捷键，帮助您更高效地创建和编辑流程图。

快捷键 功能说明
Ctrl + Z / ⌘Z 撤销上一步操作
Ctrl + Y / ⌘Y 重做撤销的操作
Delete / Backspace 删除选中的节点或连线
Ctrl + C / ⌘C 复制选中的节点
Ctrl + V / ⌘V 粘贴复制的节点
Ctrl + G / ⌘G 将选中的节点组合为子流程
Ctrl + Shift + G / ⌘Shift+G 解除节点组合
Space + 拖动 在画布上平移视图
Ctrl + + / Ctrl + - 放大/缩小画布
F2 编辑选中节点的文字标签
