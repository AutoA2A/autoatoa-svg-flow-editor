项目背景

在日常工作中，我们经常需要绘制流程图来梳理业务流程、系统架构或决策逻辑。然而，许多在线流程图工具存在以下问题：

  • 需要注册登录，流程繁琐
  • 数据上传服务器，存在隐私风险
  • 依赖网络，离线无法使用
  • 功能臃肿，学习成本高

为了解决这些问题，我们开发了这款纯浏览器端运行的流程图编辑器。它完全使用原生 Web 技术实现，无需后端服务，打开即用，关闭即走，所有数据都保存在您的设备本地。

技术栈

本项目坚持零依赖、离线可用原则，所有功能均使用原生 Web API 实现：

HTML5 Canvas / SVG 原生 JavaScript (ES6+) CSS3 自定义属性 LocalStorage API File API Drag & Drop API Web Audio API Pointer Events

无外部 CDN、无第三方框架、无构建工具。一个 HTML 文件即可运行。

使用说明

只需三步，即可开始创建您的流程图：

  1. 选择节点
    从左侧节点库中拖拽所需节点（矩形、菱形、圆角、平行四边形等）到画布中央。
  2. 连接节点
    点击节点边缘的连接点，拖拽到目标节点即可创建连线。支持直线、折线、曲线三种样式。
  3. 编辑属性
    选中任意节点或连线，在右侧属性面板中修改颜色、文字、字体大小、边框样式等。
  4. 导出保存
    完成后点击右上角"导出"按钮，选择 SVG（矢量）、PNG（图片）或 JSON（可编辑数据）格式下载。
